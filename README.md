# Translate Tool

## Installation
```bash
# Step 1
git clone https://github.com/xRiot45/translate.git

# Step 2
cd translate

# Step 3
sudo su

# Step 4
python3 -m venv .venv

# Step 5
source .venv/bin/activate

# Step 6
pip install -r requirements.txt
```

## Usage
Type ```python3 main.py``` to run this tool
